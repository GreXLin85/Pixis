# Pixis
NodeJS based advanced file downloader.
# Requirements
 - NodeJS v13.12.0
 - NPM 6.14.4
# Installation
 1. `git clone https://github.com/GreXLin85/Pixis.git`
 2. `cd Pixis`
 3. `npm install`
# Usage

    node pixis.js --link <put a link here>
# Example

    node pixis.js --link https://raw.githubusercontent.com/GreXLin85/Pixis/master/example.txt
